# 01.機能要件

このセクションでは、`DevBlueprint`が提供する、または将来提供する予定の具体的な「機能」に関する要求仕様を、機能単位で管理します。

各機能要件は、個別のMarkdownファイルとして記述され、その中で`REQ-[カテゴリ]-[番号]`形式のIDが定義されます。
